Requerimientos Funcionales y No Funcionales del Servicio de Diseño Web

Requerimientos Funcionales
  • Diseño de publicidad
  • Imagen Corporativa
  • Manejo de tipografía
  • Producción Audiovisual
  • Manejo de espacialización
  • Creación de logos
  • Manejo autogestionable

    • Requerimientos No Funcionales
  • El Software debe manejar diseños exclusivos
  • El Software debe de contar con alta calidad para sus entregas
  • El Software debe contar con una alta variedad de color y tipo de letra
